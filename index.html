Recipe Recommendation System

The Recipe Recommendation System is an innovative machine learning-powered application that leverages advanced natural language processing techniques to provide personalized recipe suggestions based on available ingredients. Read More... Developed using Python, the project integrates sophisticated technologies including scikit-learn's TF-IDF vectorization and cosine similarity algorithms to create an intelligent recommendation engine. The system processes a comprehensive Indian food dataset, transforming raw ingredient and recipe information into a sophisticated recommendation model that can match user-provided ingredients with similar and relevant recipes across various cuisines, courses, and dietary preferences.
From a user experience perspective, the application offers an intuitive graphical interface built with Tkinter, guiding users through a seamless journey of recipe discovery. Users begin at a welcoming title screen, proceed to enter their available ingredients, and are then presented with a curated list of recipe recommendations complete with detailed information such as preparation times, cuisine types, and cooking instructions. A unique feature of the application is its ability to provide direct recipe URLs, which can be easily copied to the clipboard, enhancing user convenience. The recommendation system not only suggests recipes but also provides context-rich information, making it a valuable tool for home cooks, culinary enthusiasts, and anyone looking to explore Indian cuisine or make the most of their available ingredients. Roboflow Computer Vision model Api Integration

This Python script showcases the use of the Roboflow computer vision platform to build an automated garbage detection system. The script demonstrates how to leverage Roboflow's pre-trained object detection model to identify and locate garbage in both local and remote images.
The script provides a practical example of integrating Roboflow's capabilities into a Python workflow. It shows how to set up a Roboflow project, make predictions on an image, visualize the results by overlaying bounding boxes, and save the output. This project can be used as a starting point for developers interested in building real-world computer vision applications for waste management or environmental monitoring.
