My Projects

LitterGo

My Final Year Project: AI Powered Flutter Mobile Application For Detection And Management Of Litter Dumps using Computer Vision. LitterGo aims to address the issue of litter dumps in neighborhoods. The solution involves citizens creating accounts(firebase Auth) and report by providing details and its location, capture an image of the site. The project utilizes Computer vision pretrained model's API that uses image processing for detection. Users can track the progress of their reported cases for cleanup. Management efforts over time are visualized. Information about litter dump locations, images, cleanup progress are stored in a Firebase database for efficient management and tracking. It includes google maps API, pretrained Computer vision model API from Roboflow for litter detection.

To Do List Website

The To-Do List app is a straightforward task manager designed to help users keep track of their responsibilities. After signing up or logging in, users can easily add new tasks, delete tasks they no longer need, and clear out completed ones to keep their list tidy. Each task is saved securely under their account, so they can access it from anywhere. This app combines a simple UI with essential functionality, making it perfect for anyone looking to boost their productivity without any hassle. Developed using HTML, CSS, Bootstrap, and JavaScript for creating a clean and responsive user experience.
